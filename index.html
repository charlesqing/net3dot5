DISM DeploymentDISM 部署

For IT admins deploying via SCCM, MDT, or Group Policy, use DISM with Windows installation media:

  1. Mount Windows ISO or insert installation media
  2. Open Command Prompt as Administrator
  3. Run: DISM /Online /Enable-Feature /FeatureName:NetFx3 /All /LimitAccess /Source:D:\sources\sxs
  4. Reboot if prompted

For WSUS environments, error 0x8024402C appears when Windows Update is blocked. Use /LimitAccess flag to bypass WSUS and install from local source.

对于通过 SCCM、MDT 或组策略部署的 IT 管理员，使用 DISM 配合 Windows 安装介质：

  1. 挂载 Windows ISO 或插入安装介质
  2. 以管理员身份打开命令提示符
  3. 运行：DISM /Online /Enable-Feature /FeatureName:NetFx3 /All /LimitAccess /Source:D:\sources\sxs
  4. 按提示重启

在 WSUS 环境中，Windows Update 被阻止时会报错 0x8024402C。使用 /LimitAccess 参数可绕过 WSUS 从本地源安装。
